# Instagram-clone-backend

# Install

Make sure installed [yarn](https://classic.yarnpkg.com/en/docs/install/#mac-stable) package.
`yarn install`

# How to development

1. `yarn start`
2. Open [http://localhost:5000](http://localhost:5000)

# How to build
1. `yarn build`
2. See at `dist` folder
